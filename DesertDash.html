Desert Dash

Created in collaboration with Kolten Morales for the 48-hour Neumont Spring Game Jam 2024 under the theme "Keep Moving".

A fast-paced multiplayer racing game where players must outrun a deadly desert storm while competing against each other in a race for survival.

üéÆ About the Game

Desert Dash is an intense 3D multiplayer racing game created during a 48-hour game jam. Players find themselves in a desperate race against both time and each other as they navigate through a treacherous desert landscape while being pursued by a massive, destructive storm. The game combines competitive racing mechanics with survival elements, creating a unique and thrilling multiplayer experience.

This project showcases the implementation of real-time multiplayer functionality and dynamic environmental hazards in a fast-paced racing environment.
